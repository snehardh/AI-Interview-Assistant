# 🤖 AI Interview Assistant

An AI-powered interview preparation platform that generates personalized interview questions from a candidate's **Resume** and **Job Description**, evaluates responses using **Generative AI**, and provides detailed AI-powered feedback.

---

## 🚀 Features

- 📄 Resume & Job Description Analysis
- 🤖 AI-generated Interview Questions
- ⭐ AI-powered Answer Evaluation
- 📊 Interview Progress Tracking
- 📜 Interview History
- 📥 Download Interview Report
- 🎯 Interactive Streamlit Interface
- 🔄 Multi-question Interview Flow

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/snehardh/AI-Interview-Assistant.git
```

Go to the project folder

```bash
cd AI-Interview-Assistant
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
